API pre ESP32

Pre používanie s ESP32 použite tieto endpointy:

Začať smenu pomocou ID:
POST /api/start_shift 
{
  "employee_id": 1,
  "work_location": "Zoo"
}
Začať smenu pomocou mena:
POST /api/start_shift_by_name 
{
  "employee_name": "Janko Mrkvička",
  "work_location": "Spa"
}
Ukončiť smenu pomocou ID:
POST /api/end_shift 
{
  "attendance_id": 5
}
alebo 
{
  "employee_id": 1
}
Ukončiť smenu pomocou mena:
POST /api/end_shift_by_name 
{
  "employee_name": "Janko Mrkvička"
}

Poznámka: Pri používaní mien systém hľadá zamestnanca podľa mena a priezviska. Ak zadáte iba jedno slovo, systém sa pokúsi nájsť zhobu v meni alebo priezvisku.
